=== Dashboard Features ===

1. الصفحة الرئيسية:
   - تعرض إحصائيات المنصة:
     • عدد الزيارات مع التحليل اليومي والأيام السابقة
     • عدد المستخدمين مع التحليل اليومي والأيام السابقة
     • عدد المنشورات مع التحليل اليومي والأيام السابقة

2. نظام داشبورد خاص بالمستخدم:
   - إدارة المستخدمين:
     • حذف مستخدم
     • حظر مستخدم لفترة معينة
     • تعديل بيانات المستخدم (Email, Password, Username)
   - التوثيق للمستخدم (حتى الآن بدون ميزات إضافية)
   - عرض المفضلة الخاصة بالمستخدم
   - عرض الأشخاص المحظورين للمستخدم
   - عرض سجل الأمان الخاص بالمستخدم:
     • آخر تسجيل دخول
     • آخر تحديث للبيانات
   - إنشاء منشورات باسم المستخدم
   - تحديث الإحصائيات الخاصة بالمستخدم:
     • عدد المتابعين
     • عدد الإعجابات
     • عدد الأشخاص الذين يتابعهم
   - عرض آخر خمس مرات فتح للمستخدم

3. نظام Refresh Token:
   - مفصول للواجهة الأمامية للموقع والداشبورد
   - يضمن استمرارية الجلسات بشكل آمن
